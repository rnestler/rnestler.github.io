Seting up snapcast

Following my own instructions from a previous blogpost I set up the snapcast client.

Cross Compiling snapcast

I followed again the guide from the ArchLinux distcc wiki page, just changing from the armv7 ot the armv8 toolchain:

Distcc Master System (Raspberry Pi)

sudo pacman -S distcc
sudo vim /etc/makepkg.conf
# Edit / add the following lines
BUILDENV=(distcc color !ccache check !sign)
DISTCC_HOSTS="$IP_OF_SLAVE:3636"
MAKEFLAGS="-j4"

Distcc Slave System (Desktop PC)

yay distccd-alarm-armv8 
sudo systemctl start distccd-armv8.service
